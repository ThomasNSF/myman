MyMan

MyMan is a text-mode videogame inspired by Namco's Pac-Man. It has been ported to a variety of operating systems using the following for user interaction: ncurses, PDcurses, XCurses, sysV-curses, S/Lang slcurses, Win32 console, raw stdio and termios, GGI, Allegro, aalib, libcaca, TWin, and Carbon.

Please refer to the MyMan project page at SourceForge for recent releases.

Older versions (including th original public-domain version) can still be found on bsittler's Home Page.
